# ClawHub Submission Instructions

## Manual Submission

1. Go to the ClawHub registry
2. Click "Submit Skill"
3. Fill in the following:

| Field | Value |
|-------|-------|
| Skill Directory URL | https://github.com/rankigi/openclaw-skill |
| Skill Name | rankigi-governance |
| Category | governance/security |
| Description | Governance infrastructure for OpenClaw agents. Hash-chained audit trail, behavioral monitoring, and cryptographic proof of every action. |
| Tags | governance, audit, security, hash-chain, compliance |

## Verification

After submission, the skill will appear in ClawHub search results under "rankigi-governance". Users can install directly from the registry.
